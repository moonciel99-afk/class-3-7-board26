🏫 학급 생활 안내

학급 규칙과 생활 안내를 확인하고 즐거운 학교생활을 만들어가요

📜 학급 규칙 ▼
  • 수업 시작 5분 전까지 착석하여 수업을 준비합니다
  • 휴대폰은 등교 시 보관함에 제출하고 수업 중에는 사용하지 않습니다
  • 청소 당번표를 준수하여 깨끗한 교실 환경을 만듭니다
  • 서로 존중하는 언어를 사용하고 친구들을 배려합니다
  • 학교 시설물을 소중히 다루고 공공질서를 지킵니다
  • 지각, 조퇴, 결석 시 반드시 담임 선생님께 사전 연락합니다
📱 휴대폰 사용 규정 ▼
  • 등교 시 교실 앞 휴대폰 보관함에 제출합니다
  • 점심시간(12:30~13:00) 및 방과후에 사용 가능합니다
  • 수업 중 휴대폰 사용이 적발될 경우 즉시 수거됩니다
  • 긴급한 연락이 필요한 경우 담임 선생님께 말씀드리고 허락을 받습니다
  • 휴대폰으로 타인을 촬영하거나 사생활을 침해하는 행위는 금지됩니다
📝 과제 제출 방법 ▼
  • 과제는 제출 기한을 반드시 지켜주세요 (지각 제출 시 감점 가능)
  • 온라인 제출: Google Classroom 또는 각 과목 선생님께서 안내하신 플랫폼을 이용합니다
  • 오프라인 제출: 담당 교과 선생님께 직접 제출하거나 과제함에 넣어주세요
  • 과제 제출이 어려운 경우 사전에 담당 선생님께 말씀드리세요
  • 표절이나 대리 제출은 절대 금지이며, 적발 시 0점 처리됩니다
💬 상담 신청 방법 ▼
  • 담임 선생님 상담: 점심시간(12:30~13:00) 또는 방과후 시간 이용
  • 진로 상담: 매주 화요일, 목요일 방과후 진로상담실에서 진행
  • 온라인 상담 신청: 아래 버튼을 클릭하여 상담 신청서를 작성해주세요
  • 긴급 상담이 필요한 경우 언제든지 담임 선생님께 연락주세요
  • 모든 상담 내용은 비밀이 보장됩니다
